Apple相关问题总结

  • 固件下载： 爱思助手或者 ipsw.me

  • iTunes下载的固件所在位置：C:\Users\你的用户名\AppData\Roaming\Apple Computer\iTunes\iPhone Software Updates

  • iTunes备份所在位置：C:\Users\用户名\AppData\Roaming\Apple Computer\MobileSync\Backup

  • 用itunes更新ios14失败，“发生未知错误(4000)”。解决：提前下载好固件，在iTunes中按住shift同时点击“更新”

随手记

  • 2020.9.29
    用路由器登录校园网的运营商账号时，出现“运营商用户认证失败!失败原因[brac error: create session failed(mac collision)!]”的问题，重启路由器等各种方法试了都不行，感觉是移动公司故意针对这种用路由器登账号的行为。困惑中突然发现，最后的mac collision意思是Mac地址冲突，去路由器设置里修改一下Mac地址就奇迹般地好啦！可以正常登陆账号了！ 回到顶部
