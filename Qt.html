历史记录：此功能非常重要！！ 首选。一定要调出来（窗口-历史记录）。

  • 证件照改背景：
    1. 点击左侧“快速选择工具(W)”（和魔棒工具在一起）
    2. 点击上方的“选择主体”，点击“选择并遮住”
    3. 在右侧找到输出设置，勾选“净化颜色”，确定。
    4. 新建一个图层，把背景色改为想要的颜色，然后Ctrl+Delete填充颜色。
    5. 把新的纯色图层放下层即可。
  • 各种报错解决：
    1. QT编译错误——cannot find file .proCSDN
    2. 点击上方的“选择主体”，点击“选择并遮住”
    3. 在右侧找到输出设置，勾选“净化颜色”，确定。
    4. 新建一个图层，把背景色改为想要的颜色，然后Ctrl+Delete填充颜色。
    5. 把新的纯色图层放下层即可。
