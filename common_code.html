Windows常用代码

  • 获得当前系统的Key
    powershell "(Get-WmiObject -query 'select * from SoftwareLicensingService').Oa3xOriginalProductKey"

  • 卸载key，至未激活状态
    slmgr.vbs -upk

  • 查看win何时过期
    slmgr.vbs -xpr

  • 查看详细激活信息
    slmgr.vbs -dlv

  • 3600s后自动关机
    shutdown -s -t 3600

  • 取消自动关机
    shutdown -a

  • 校验文件
    CertUtil -hashfile （文件路径) （算法）
    例如：CertUtil -hashfile D:\123.exe MD5 校验文件MD5值

  • 激活visio

  • KMS激活win
    slmgr /ipk THE_CLIENT_KEY
    slmgr /skms YOUR_KMS_IP
    slmgr /ato

    Win10专业版（半年一激活） W269N-WFGWX-YVC9B-4J6C9-T83GX
    Win10企业版（半年一激活） NPPR9-FWDCX-D2C8J-H872K-2YT43

  • 查看OFFICE系列激活状态

    cscript "C:\Program Files\Microsoft Office\Office16\ospp.vbs" /dstatus
    cscript "C:\Program Files (x86)\Microsoft Office\Office16\ospp.vbs" /dstatus
